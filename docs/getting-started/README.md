# 快速开始

## 简介

ZAN Blockchain Skills 面向 **Agent** 与 **开发者** 提供标准化的区块链能力封装，帮助您更高效地完成链上查询、数据分析、交易辅助与运维管理等任务。

## 接入方式

当前支持通过 **API Key** 方式接入 ZAN 区块链服务。

| 方式 | 适用场景 | 特点 |
|------|----------|------|
| **API Key** | 企业客户、高频调用、需要完整功能 | 需注册 ZAN 账户，功能完整，支持订阅套餐 |
