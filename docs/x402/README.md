# x402 支付

> **注意：x402 支付功能正在开发中，暂未开放。以下内容为预览版本，待功能上线后正式启用。**

## 概述

x402 Payment 为 AI Agent、低频用户和隐私敏感用户提供「即用即付」的区块链服务访问能力，**无需注册账户**。通过链上 USDC 即时支付，即可调用 ZAN 的 Chain RPC、Data 等能力。

## 什么是 x402

x402 是基于 HTTP **402 Payment Required** 状态码的开放支付标准。当服务端需要收费时，返回 402 状态码及支付指令，客户端完成链上支付后即可获取资源。

### 工作流程

```
┌─────────────┐                    ┌─────────────┐
│   客户端     │                    │   服务端     │
└──────┬──────┘                    └──────┬──────┘
       │                                  │
       │  1. 请求资源                      │
       │ ─────────────────────────────────>
       │                                  │
       │  2. 402 Payment Required         │
       │     (含支付指令)                   │
       │ <─────────────────────────────────
       │                                  │
       │  3. 准备并提交链上支付             │
       │ ─────────────────────────────────>
       │                                  │
       │  4. 验证支付，返回结果             │
       │ <─────────────────────────────────
       │                                  │
```

1. **客户端请求资源**：发起 RPC 或 Data API 请求
2. **服务端返回 402**：响应中包含支付金额、收款地址、签名等支付指令
3. **客户端提交支付**：在链上完成 USDC 转账
4. **服务端验证并返回**：验证支付成功后返回请求结果

## 接入方式

1. 安装 x402 SDK（如 `x402-axios`）
2. 配置钱包私钥（用于签名支付）
3. 通过 SDK 发起请求，SDK 会自动处理 402 支付流程

详见 [x402 SDK 使用指南](./x402-sdk-guide.md)。

## 定价模型

| 网络类型 | 价格 | 信用额度 | 说明 |
|---------|------|---------|------|
| 正式网络 | $0.15 USDC | 1M credits | Base Mainnet |
| 测试网络 | $0.10 USDC | 0.1M credits | Base Sepolia |

- **默认信用包**：10,000 credits
- **正式网络**单次最小支付：$0.0015
- **测试网络**单次最小支付：$0.01

## 典型场景

| 场景 | 说明 |
|------|------|
| AI Agent 自动调用 | Agent 无需人工注册，自动完成支付并获取数据 |
| 低频用户 | 偶尔查询，无需订阅或预充值 |
| 隐私敏感用户 | 无需提供邮箱、手机等个人信息 |
| 快速原型开发 | 5 分钟接入，快速验证想法 |

## Agent 使用建议

在以下情况下推荐使用 x402：

- 用户**没有 ZAN 账户**
- 用户**不愿提供个人信息**
- 需要**快速测试**或原型验证
- 您是 **AI Agent 开发者**，希望 Agent 自主完成支付
